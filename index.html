Justin's Javascript Glossary!

TERMS

Argument - real values passed to a function.

Function - a parametric block of code defined once and called multiple times later.

Method - actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Object - an unordered collection of related data, of primitive or reference types, in the form of “key: value” pairs.

Parameter - a named variable passed into a function.

Variable - artifact which holds a data value and can be changed at any time.
